## 如何使用

### 安装
1. 插件已在当前目录编译完成
2. 确保 Obsidian vault 路径正确
3. 重启 Obsidian 或重新加载插件

### 使用步骤
1. 在编辑器中选中要朗读的文本
2. 打开命令面板（Ctrl+P 或 Cmd+P）
3. 执行"朗读选中文本"命令
4. 听到语音朗读，状态栏显示"播放中"

### 设置
1. 打开设置 → 插件选项 → 找到本插件
2. 在"EdgeTTS 设置"区域：
   - 选择喜欢的语音（晓晓、云希等）
   - 调整语速（向右加速，向左减速）
   - 设置最大文本长度（默认1800字符）

### 播放控制
- **朗读选中文本** - 开始朗读
- **停止朗读** - 立即停止
- **暂停/继续朗读** - 在播放和暂停之间切换
