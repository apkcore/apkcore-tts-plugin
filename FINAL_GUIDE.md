# 🎯 TTS 插件 - 最终使用指南

## ✨ 统一的智能朗读体验

**所有操作方式现在都支持智能判断：**
- ✅ **有选中文本** → 朗读选中内容
- ✅ **无选中文本** → 朗读当前文档全文

---

## 🚀 四种操作方式（全部智能）

### 1️⃣ 快捷键 Ctrl+R（最快）
```
场景1：朗读全文
1. 打开文档（不选中任何文字）
2. Ctrl+R
3. ✅ 朗读整篇文档

场景2：朗读选中段落
1. 选中一段文字
2. Ctrl+R
3. ✅ 只朗读选中部分
```

### 2️⃣ 右键菜单（最直观）
```
场景1：朗读全文
1. 打开文档
2. 右键空白处
3. 选择 "🔊 朗读全文"
4. ✅ 朗读整篇文档

场景2：朗读选中段落
1. 选中文字
2. 右键
3. 选择 "🔊 朗读选中文本"
4. ✅ 只朗读选中部分
```

### 3️⃣ 侧边栏图标（快速访问）
```
场景1：朗读全文
1. 打开文档
2. 点击左侧 🔊 图标
3. ✅ 朗读整篇文档

场景2：朗读选中段落
1. 选中文字
2. 点击左侧 🔊 图标
3. ✅ 只朗读选中部分
```

### 4️⃣ 悬浮停止按钮（随时停止）
```
不管用哪种方式开始朗读：
1. 开始朗读后，右下角自动出现 ⏹️
2. 不想听了？点击 ⏹️
3. ✅ 立即停止
```

## 🎯 推荐使用场景

### 📖 场景1：快速浏览长文档
```
操作：
1. 打开文档
2. Ctrl+R（或点 🔊）
3. 边听边看，快速了解内容
4. 点 ⏹️ 停在感兴趣的地方

优势：
✅ 无需全选，直接朗读
✅ 解放双眼，多任务处理
✅ 提高效率
```

### ✏️ 场景2：校对文章
```
操作：
1. 写完文章
2. Ctrl+R 朗读全文
3. 听出语句不通顺的地方
4. 点 ⏹️ 停下修改

优势：
✅ 发现视觉盲区的错误
✅ 检查语句流畅度
✅ 提升文章质量
```

### 📝 场景3：精读重点段落
```
操作：
1. 选中重点段落
2. Ctrl+R
3. 反复听几遍加深理解

优势：
✅ 聚焦重点内容
✅ 加深记忆
✅ 提高学习效率
```

### 🌙 场景4：睡前听笔记
```
操作：
1. 打开复习笔记
2. 点击 🔊 朗读全文
3. 躺下放松听
4. 困了就点 ⏹️（或让它读完）

优势：
✅ 解放双手
✅ 放松学习
✅ 睡前复习
```

## 💡 高级技巧

### 技巧1：分段精读长文
```
1. Ctrl+R 快速听一遍全文（了解大意）
2. 选中难懂的段落
3. Ctrl+R 反复听
4. 理解透彻
```

### 技巧2：边写边听
```
1. 写完一段
2. Ctrl+R 听一遍
3. 发现问题立即修改
4. 持续优化文章
```

### 技巧3：多任务学习
```
1. 打开学习笔记
2. Ctrl+R 开始朗读
3. 同时做其他事（整理资料、画图等）
4. 用听觉吸收知识
```

### 技巧4：调整语速适应不同场景
```
快速浏览：+40% ~ +60%（设置 → 插件选项）
正常阅读：0% ~ +20%
精读学习：-20% ~ 0%
```

## 🎨 界面元素

### 左侧边栏
- **🔊 图标** - 智能朗读按钮
- 位置：左侧工具栏
- 点击：智能判断朗读全文或选中

### 右下角悬浮按钮
- **⏹️ 停止按钮** - 圆形红色
- 位置：右下角
- 大小：60×60 像素
- 显示：朗读时自动出现
- 隐藏：停止/完成后自动消失

### 底部状态栏
- **TTS: 状态** - 实时反馈
- 状态：就绪 / 正在合成... / 播放中... / 已停止

### 右键菜单
- **🔊 朗读选中文本** - 选中文字时显示
- **🔊 朗读全文** - 未选中时显示

## 🎯 最佳实践

### ✅ 推荐做法
- 经常使用 Ctrl+R（最快）
- 利用右下角 ⏹️ 随时停止
- 根据场景调整语速
