# 🎯 Obsidian TTS 插件 - 快速使用指南

## ✨ 新增易用性功能

### 1. **快捷键**（最快）
- **Ctrl+R** - 朗读选中文本
- **Ctrl+E** - 停止朗读
- ~~**Ctrl+P** - 暂停/继续~~ (可能与命令面板冲突，建议在设置中自定义)

**使用方法：**
1. 选中文本
2. 按 `Ctrl+R`
3. 立即朗读！

### 2. **右键菜单**（最直观）
1. 选中文本
2. 右键点击
3. 选择 "🔊 朗读选中文本"

### 3. **侧边栏图标**（快速访问）
- 左侧边栏有一个 **🔊** 图标
- 点击后朗读当前选中的文本

### 4. **状态栏**（实时反馈）
- 底部状态栏显示当前状态：
  - `TTS: 就绪 (浏览器TTS)`
  - `TTS: 正在合成...`
  - `TTS: 播放中...`
  - `TTS: 已暂停`

## 🚀 推荐使用方式

### 场景 1：阅读笔记
```
1. 选中一段文字
2. 按 Ctrl+R（或右键选择朗读）
3. 边听边看，加深理解
```

### 场景 2：校对文章
```
1. 选中需要校对的段落
2. 按 Ctrl+R
3. 听出语句不通顺的地方
```

### 场景 3：学习外语
```
1. 选中外语文本
2. 按 Ctrl+R
3. 听标准发音（取决于系统TTS）
```

## ⚙️ 自定义快捷键

如果默认快捷键冲突：
1. 打开 **设置 → 快捷键**
2. 搜索 "TTS" 或 "朗读"
3. 点击命令右侧的按钮
4. 按下你想要的快捷键
5. 保存

**推荐快捷键组合：**
- `Ctrl+Shift+R` - 朗读
- `Ctrl+Shift+E` - 停止
- `Ctrl+Shift+P` - 暂停/继续

## 🎛️ 调整语速

在 **设置 → 插件选项 → EdgeTTS 设置**：
- 向左拖动滑块 = 变慢
- 向右拖动滑块 = 变快
- 0 = 正常速度
- +20% = 建议的快速阅读速度
- -20% = 建议的慢速学习速度

## 💡 使用技巧

### 技巧 1：快速浏览长文
1. 调整语速到 +30% ~ +50%
2. 逐段选中并朗读
3. 快速了解文章大意

### 技巧 2：深度学习
1. 调整语速到 -20% ~ 0%
2. 朗读重要段落
3. 配合笔记加深记忆

### 技巧 3：多任务处理
1. 朗读笔记内容
2. 同时做其他事情
3. 用听觉替代视觉

### 技巧 4：检查错别字
1. 朗读自己写的文章
2. 听到奇怪的发音 = 可能有错别字
3. 快速定位并修改

## 🐛 常见问题

### Q: 为什么按 Ctrl+R 没反应？
**A:** 请先选中文本，再按快捷键。

### Q: 如何更改快捷键？
**A:** 设置 → 快捷键 → 搜索 "朗读" → 重新设置。

### Q: 为什么找不到右键菜单？
**A:** 必须先选中文本，右键菜单才会出现 "朗读" 选项。

### Q: 语音听起来很机械？
**A:** 这是系统自带的 TTS 引擎决定的。可以：
- Windows: 安装微软语音包
- macOS: 在系统偏好设置中下载更多语音
- 或等待 EdgeTTS 功能修复（更好的音质）

### Q: 可以朗读整篇文章吗？
**A:** 可以全选文本（Ctrl+A）再朗读，但注意：
- 默认最大长度 1800 字符
- 可在设置中调整
