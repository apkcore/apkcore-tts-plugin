# 🎉 TTS 插件更新说明

## ✨ 新功能

### 1. 一键朗读全文
**侧边栏图标智能判断：**
- ✅ **有选中文本** → 朗读选中内容
- ✅ **无选中文本** → 朗读当前文档全文

**使用场景：**
```
场景1：朗读整篇笔记
1. 打开笔记
2. 点击侧边栏 🔊 图标
3. 直接朗读全文，无需全选！

场景2：朗读特定段落
1. 选中某段文字
2. 点击侧边栏 🔊 图标
3. 只朗读选中部分
```

### 2. 悬浮停止按钮 ⏹️
**位置：** 右下角
**样式：** 圆形悬浮按钮，红色背景
**显示时机：** 开始朗读时自动出现
**隐藏时机：** 
- 点击停止按钮
- 播放完成
- 出现错误

**特点：**
- 🎯 大按钮，容易点击
- ✨ 鼠标悬停时放大
- 🌊 平滑动画效果
- 🎨 自适应主题颜色

**使用方法：**
```
1. 开始朗读（Ctrl+R 或点击图标）
2. 右下角自动出现 ⏹️ 按钮
3. 想停就点，一键搞定！
```

## 🎯 完整的使用流程

### 快速朗读全文
```
方式1（最快）：
1. 打开文档
2. 点击左侧边栏 🔊
3. 开始朗读
4. 右下角出现停止按钮
5. 不想听了就点 ⏹️

方式2（快捷键）：
1. 打开文档
2. Ctrl+A（全选）
3. Ctrl+R（朗读）
4. Ctrl+E（停止）
```

### 朗读选中段落
```
方式1（右键菜单）：
1. 选中文本
2. 右键 → "🔊 朗读选中文本"
3. 点右下角 ⏹️ 停止

方式2（快捷键）：
1. 选中文本
2. Ctrl+R
3. Ctrl+E 停止
```

## 💡 使用技巧

### 技巧 1：快速浏览长文档
```
1. 打开长文档
2. 点击 🔊 朗读全文
3. 边听边看，快速浏览
4. 听到感兴趣的部分 → 点 ⏹️ 停下
5. 仔细阅读该部分
```

### 技巧 2：多任务工作
```
1. 打开需要复习的笔记
2. 点击 🔊 开始朗读
3. 同时处理其他事情（整理文件、写代码等）
4. 用听觉吸收信息
5. 随时点 ⏹️ 暂停
```

### 技巧 3：校对文章
```
1. 写完文章后，点击 🔊 朗读全文
2. 听是否有语句不通顺
3. 发现问题 → 点 ⏹️
4. 修改 → 再次朗读
5. 直到满意为止
```

### 技巧 4：学习资料
```
1. 导入学习材料到 Obsidian
2. 调整语速（设置 → 插件选项）
3. 点击 🔊 开始学习
4. 累了就点 ⏹️ 休息
```

## 🔧 自定义建议

### 推荐快捷键设置
```
朗读：Ctrl+R        （默认）
停止：Ctrl+E        （默认）
暂停：Ctrl+Shift+P  （需自己设置）
```

### 推荐语速设置
```
快速浏览：+30% ~ +50%
正常阅读：0% ~ +20%
仔细学习：-20% ~ 0%
```
